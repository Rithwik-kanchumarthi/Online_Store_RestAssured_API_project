# Online Store REST Assured API Automation Framework

## Overview
This project is a REST API Automation Framework built using Java and Rest Assured. It follows industry-standard automation practices with Maven, TestNG, POJO classes, and reporting tools to ensure scalable, maintainable, and reliable API testing.

## Tech Stack
- Java
- Rest Assured
- Maven
- TestNG
- POJO (Plain Old Java Objects)
- Extent Reports
- Allure Reports
- Git & GitHub
- Jenkins (CI/CD)

## Project Features
- REST Assured Framework Development
- Maven Project with Standard Folder Structure
- API Endpoint (Routes) Management
- Schema and Model Understanding
- POJO Class Implementation
- Request Payload Creation
- Configuration and Test Data Management
- API Test Creation with Assertions and Validations
- Extent Report Generation
- Allure Report Integration
- Test Execution using TestNG
  - Serial Execution
  - Parallel Execution
- Continuous Integration using Jenkins
- Version Control using Git & GitHub
